배열정의

  • 순서가 존재하는 테이터의 창고
  • 대괄호로 생성[]
  • 각각의 요소는 쉼표로 구분
  • 0부터 시작하는 위치 데이터 index 0.1.2.3
    • =>이를 활용해서 요소에 접근

배열속성

length

배열에 존재하는 요소의 개수를 기반으로 해당배열의 길이를 담은 속성 1.2.3.4

배열메서드

  • .push() 데이터 추가
  • .pop() 데이터 삭제
  • .indexOf index 조회
  • .includes() 포함여부 확인

객체 정의

  • 중괄호를 사용해서 정의{}
  • 내부의 요소는 프로퍼티 (키key + 값 value)
    • Dot notation
    • Bracket notation
      • 변수를 통해서 객체 프로퍼티에 접근 가능한 방법
  • 위 두가지로 새로운 프로퍼티도 생성

객체 메서드

  • Object.keys()
    • key 모음을 배열로 ("문자열로")
  • Object.values()
    • value 모음을 배열로 (데이터 그대로)
