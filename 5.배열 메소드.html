배열(Array)

순서가 존재하는 여러개의 데이터를 관리해야할때 배열을 활용
여러개의 데이터를 순서와 함께 저장하는 창고
["제이슨","엘리스","크리스","제니","톰"]
각 요소 (Element)는 index라는 값을 가진다 index 해당 요소의 순서 0,1,2,3,4

배열에 사용할 수 있는 기능이 담긴 명령 Method

  • push()
    • 배열의 가장 뒤에 데이터를 추가
  • pop()
    • 배열 가장 끝에 있는 데이터 제거
  • indexOf()
    • 특정 배열에서 주어진 데이터 index 값을 찾아 반환
  • includes()
    • 특정 배열에 주어진 데이터가 존재하는지 여부 확인 여부에 따라 참(ture),거짓(false)반환
