배열(Array)

순서가 존재하는 여러개의 데이터를 관리해야할때 배열을 활용
여러개의 데이터를 순서와 함께 저장하는 창고
["제이슨","엘리스","크리스","제니","톰"]
각 요소 (Element)는 index라는 값을 가진다 index 해당 요소의 순서 0,1,2,3,4

배열이 가지고 있는 속성 Property

  • 특정 배열의 길이를 나타내는 속성
    • 길이(length) => 배열 내 요소의 개수
    length는 index와 다르게 1부터 시작함

배열에 사용할 수 있는 기능이 담긴 명령 Method
